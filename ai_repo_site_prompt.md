# PROMPT ДЛЯ ИИ: СОЗДАТЬ САЙТ С НУЛЯ В РЕПОЗИТОРИИ

Твоя задача: создать с нуля готовый сайт в репозитории для B2B-продукта, связанного с распределёнными командами.

## 1. Что это за продукт

Это НЕ event-агентство и НЕ сайт “корпоративов”.

Это продукт для работы с распределёнными и гибридными командами через игровые и иммерсивные форматы.

Ключевая идея продукта:
- улучшать связи внутри команды
- ускорять онбординг
- снижать выпадение удалённых сотрудников
- давать полезный сигнал о состоянии команды
- помогать HR и руководителям через формат, который не выглядит душным

Формула продукта:
**Experience → Dynamics → Signal**
или:
**Событие → Поведение → Понимание команды**
